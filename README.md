# BlueSaNGG.github.io

Main GitHub Pages repository for **https://bluesangg.github.io/**.

## Purpose

This repo is the **front door** of the personal site.
It should stay focused on a small number of high-level entry pages.

## What this repo is for

Use this repo for:
- the root landing page
- the interactive resume page
- light personal-site navigation
- assets needed by the root site

Do **not** use this repo for large course portfolios or many artifact pages unless they truly belong on the root site.

## Main pages

- `/` → landing page
- `/interactive-resume.html` → preserved interactive resume style page

## Relationship to other site repos

This repo works together with the separate portfolio repo:
- `BlueSaNGG/CV` → publishes to `https://bluesangg.github.io/CV/`

Recommended division of responsibility:
- `BlueSaNGG.github.io` = homepage / routing / personal brand entry
- `CV` = detailed portfolio pages and artifact pages

## Structure rule

If a page answers the question **"Where should I start?"**, it probably belongs here.
If a page answers **"Show me the full portfolio / artifact details"**, it probably belongs in the `CV` repo.

## Notes

- This repo is intentionally small so the root site stays easy to maintain.
- If GitHub Pages seems stale after a push, wait for deployment/cache refresh before assuming the change failed.
